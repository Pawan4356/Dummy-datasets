# 🛡️ SafeSpace - Real-time Safety Intelligence Platform

SafeSpace is a comprehensive real-time safety intelligence platform that displays threat data across India using interactive heatmaps and provides AI-generated safety advice.

## ✨ Features

### 🔹 Interactive Dashboard
- **Real-time Heatmap**: Visual representation of threat intensity across Indian cities
- **Threat Feed**: Dynamic list of safety threats with detailed information
- **AI Safety Advice**: Personalized safety recommendations generated by AI
- **Dark/Light Mode**: Toggle between themes for better user experience

### 🔹 Modern UI Components
- **Glassmorphism Design**: Beautiful translucent effects with backdrop blur
- **Responsive Layout**: Works seamlessly on desktop and mobile devices
- **Animated Interactions**: Smooth transitions and hover effects
- **Professional Navigation**: Clean header with search functionality

### 🔹 Smart Features
- **Location Detection**: Automatically detects user's location using IP
- **City Search**: Search and switch between different Indian cities
- **Threat Severity Classification**: Automatic categorization of threats (High/Medium/Low)
- **Real-time Updates**: Live threat data from news sources

## 🚀 Tech Stack

### Frontend
- **React.js 19.1.0** - Modern UI framework
- **Tailwind CSS 4.1.11** - Utility-first CSS framework
- **Leaflet.js 1.9.4** - Interactive maps
- **React-Leaflet 5.0.0** - React components for Leaflet
- **Framer Motion** - Animation library
- **React Icons** - Icon library

### Backend
- **FastAPI** - Modern Python web framework
- **Uvicorn** - ASGI server
- **OpenRouter API** - AI-powered safety advice generation
- **NewsAPI** - Real-time news data
- **IPInfo API** - User location detection

## 🛠️ Installation & Setup

### Prerequisites
- Node.js 18+ and npm
- Python 3.8+
- Git

### Backend Setup
```bash
# Navigate to backend directory
cd backend

# Install dependencies
pip install --break-system-packages -r requirements.txt

# Start the server
python -m uvicorn app:app --host 0.0.0.0 --port 8000 --reload
```

### Frontend Setup
```bash
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start the development server
npm start
```

## 🔧 Configuration

### API Keys Setup
The application uses several APIs that require keys:

1. **NewsAPI**: For fetching real-time news data
2. **OpenRouter**: For AI-generated safety advice
3. **IPInfo**: For location detection

### Environment Variables
Create a `.env` file in the backend directory:
```bash
NEWSAPI_KEY=your_newsapi_key_here
OPENROUTER_API_KEY=your_openrouter_key_here
```
